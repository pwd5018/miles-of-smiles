# Miles of Smiles

A mobile-first family road-trip companion with random games, scavenger hunts,
conversation prompts, favorites, and completed-game tracking.

## Deploy to Vercel

1. Extract this ZIP.
2. Go to https://vercel.com/drop.
3. Drag the extracted `miles-of-smiles` folder onto the page.
4. Choose a project name and click **Deploy**.

## Run locally

Install Node.js, then run:

```bash
npm install
npm run dev
```
